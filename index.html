Yile System Dynamics项目介绍

Yile System Dynamics气候模拟器是河南省太能控股推出的一款基于国际先进气候模型的本地化气候政策模拟工具。该项目旨在为政策制定者、教育工作者、企业和公众提供一个直观的平台，用于测试和探索不同气候政策的影响。

我们与国际气候研究机构合作，将全球领先的气候模拟技术与中国国情相结合，开发出这款适合中国用户的气候模拟工具。通过Yile System Dynamics，用户可以探索各种政策选择对能源结构、温室气体排放和全球温度的影响，从而为制定更有效的气候政策提供科学依据。

Yile System Dynamics 使用指南

📖

基本操作说明

欢迎使用Yile System Dynamics气候模拟器，本指南将帮助您了解如何使用该工具探索不同气候政策的影响。

1

模拟器界面概述

Yile System Dynamics模拟器界面主要由三部分组成：顶部的温度预测显示、中部的图表区域和底部的参数调节区域。温度预测显示实时反映您所设置的政策组合对2100年全球温度的影响；图表区域展示能源结构和温室气体排放的变化趋势；参数调节区域允许您调整各种政策参数。

2

参数调节滑块使用方法

通过移动滑块，您可以调整不同政策的力度。向右移动表示加强政策干预（增加百分比），向左移动表示减弱政策干预（减少百分比）。每个滑块旁边都有状态指示，显示当前设置相对于基准情景的具体变化百分比。

3

图表解读方法

左侧的能源结构图表显示不同能源类型在总能源供应中的占比变化趋势，右侧的温室气体排放图表显示净排放量的变化趋势。图表横轴为时间（2000-2100年），纵轴分别为能源供应量（EJ/年）和温室气体排放量（GtCO2e/年）。

4

情景保存与分享功能

点击顶部导航栏的"分享您的场景"按钮，可以生成一个包含您当前所有参数设置的链接，方便您保存或与他人分享您的模拟结果。

使用小贴士

尝试先调整单个参数观察其影响，然后再组合多个参数，这样可以更好地理解不同政策之间的相互作用。

🔧

主要功能模块说明

1

能源供应模块

包括煤炭、石油、天然气、可再生能源、核能和生物能源等参数调节，允许您模拟不同能源政策对能源结构的影响。例如，您可以通过提高可再生能源补贴或对化石燃料征税来改变能源结构。

2

交通与建筑模块

包括能源效率和电气化两个关键参数，允许您模拟提高终端用能效率和推广电气化对能源消费和排放的影响。

3

碳管理模块

包括碳价格和碳捕获两个参数，允许您模拟碳定价政策和碳捕获技术的应用对减排的影响。

4

温度预测显示与解读

顶部的温度预测显示实时反映您所设置的政策组合对2100年全球温度上升的影响。温度以摄氏度为单位，表示相对于工业化前水平的温度上升。

❓

常见问题解答

如何理解温度预测结果？

温度预测结果表示在您设置的政策组合下，到2100年全球平均温度相对于工业化前水平的上升幅度。根据《巴黎协定》，国际社会的目标是将全球温度上升控制在2°C以内，并努力限制在1.5°C以内。

如何比较不同政策组合的效果？

您可以先设置一组政策参数，记录温度预测结果，然后调整参数设置另一组政策，再比较两组政策下的温度预测结果和能源结构、排放趋势的差异。

如何保存和分享我的模拟结果？

点击顶部导航栏的"分享您的场景"按钮，系统会生成一个包含您当前所有参数设置的链接，您可以保存这个链接或分享给他人。

模拟器数据更新频率是多少？

我们会定期更新模拟器的基础数据和模型参数，以反映最新的科学研究成果和全球气候政策变化。每次更新都会在"数据来源"页面公布。

Yile System Dynamics 数据来源

📊

核心数据来源

Yile System Dynamics气候模拟器基于权威科学数据和模型，本页面详细说明了我们使用的数据来源和模型方法。

🌍

IPCC报告

我们的气候物理模型和温度预测算法主要基于IPCC第六次评估报告(AR6)的科学发现和模型参数。IPCC是全球气候变化研究的权威机构，其报告代表了国际科学界对气候变化的共识。

⚡

IEA数据

我们的能源系统模型和基准情景参数主要来自IEA的《世界能源展望》和《能源技术展望》报告。这些数据包括全球和区域能源供需、能源技术成本和性能、能源政策等信息。

🇨🇳

中国国家统计局和能源局数据

为了更好地反映中国国情，我们使用了中国国家统计局和国家能源局发布的能源统计数据，包括能源生产和消费结构、碳排放数据等。

🌱

全球碳计划

我们的碳循环模型和历史排放数据主要来自全球碳计划的年度碳预算报告，该报告提供了全球和区域温室气体排放的权威数据。

🧮

模型方法学

🔄

系统动力学方法

Yile System Dynamics采用系统动力学方法构建模型，这种方法特别适合模拟复杂系统中的反馈循环和时间延迟，能够更好地捕捉能源-经济-气候系统的复杂动态。

🔗

集成评估模型框架

我们的模型基于集成评估模型(IAM)框架，将能源系统、经济系统和气候系统集成在一起，能够评估不同政策对这三个系统的综合影响。

🌐

多区域模型结构

模型将全球分为多个区域，包括中国、美国、欧盟、印度等主要经济体和排放国，每个区域都有其特定的能源结构、经济特征和减排潜力。

⚖️

政策参数校准

模型中的政策参数经过校准，确保其反映真实世界中政策干预的效果。校准过程使用了历史数据和专家判断，并通过敏感性分析验证了参数的合理性。
